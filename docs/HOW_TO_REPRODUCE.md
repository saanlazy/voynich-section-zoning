# How to Reproduce

1. Install Python dependencies from `requirements.txt` if available.
2. Run `python scripts/paper6_section_zoning_pipeline.py`.
3. Run `python scripts/paper6_advanced_experiments.py`.
4. Run `python scripts/update_section_classification_models.py` if refreshing classification results.

Random seed: 42. Permutation/bootstrap/shuffle N: 1000 where used. See `outputs/qc/` for validation notes.
